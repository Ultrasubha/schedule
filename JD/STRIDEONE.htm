About Company

StrideOne is a tech-led financial services platform for startups, from the founders of India’s award-winning venture debt fund Stride Ventures. The company focuses on providing smartly designed customized financial solutions to MSMEs and supply chain partners of startups (anchors). This enables a win-win distribution model that provides access to intelligently customised growth capital for MSMEs and also enables risk efficient penetration into untapped borrower segments for StrideOne’s lending partners.

Job ProfilePython Intern
Job Overview: 
** We are seeking a Python Intern to become part of our IT team. The candidate should have a foundational understanding of Python programming and experience in project development. We appreciate interns with strong logical and analytical skills and a passion for coding and technology.

Responsibilities:
** Assist with developing, testing, and implementing new software programs using Python under the supervision of our senior developers.
** Collaborate with other developers to design and optimize code.
** Assist in designing and updating software databases using SQL or NoSQL databases.
** Participate in analyzing user needs and developing software solutions.
** Assist in maintaining documentation of program development and revisions.

Requirements:
** Foundational understanding of Python and its associated libraries.
** Experience in developing web applications using Python frameworks such as Django or Flask is a plus.
** Understanding of fundamental design principles behind a scalable application.
** Ability to work independently or with a team and multi-task effectively.
** Basic knowledge of SQL/NoSQL databases.
** Experience using version control systems such as Git is a plus.
** Proven logical and analytical problem-solving abilities.
** Prior experience working on projects on repositories like GitHub is a plus.
 
Java Intern
Job Overview: 
** We are looking for a Java Intern to join our IT team. The candidate should be a recent graduate with a foundational understanding of Java and experience with project development. We value team players with strong problem-solving skills and a keen interest in technology.

Responsibilities:
** Assist with the design, development, and maintenance of Java applications under the guidance of our senior developers.
** Participate in software analysis, programming, testing, and debugging.
** Help ensure design compliance with specifications.
** Assist in preparing and producing releases of software components.
** Support continuous improvement initiatives by exploring alternatives and technologies, and presenting these for architectural review.

Requirements:
** Foundational knowledge of Java programming, with a focus on web application development.
** Familiarity with the software development lifecycle.
** Basic understanding of database design and knowledge of SQL.
** Basic knowledge of object-oriented programming and application development tools using Java frameworks.
** Excellent problem-solving skills and a passion for technical challenges.
** Must be self-motivated, capable of working independently or part of a team.
** Prior experience working on projects on repositories like GitHub is a plus.
