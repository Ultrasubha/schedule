Job Profile

Job Profile :
** The Intern DET (Development Engineer in Test) is responsible for developing and executing comprehensive suites of automation of test plans and test cases, maintaining, and adding to the existing automation framework, automation for newly integrated features, and development of toolsets to augment SDET and Development testing.
**  The Test Engineer will also work closely with Software Developers and other SDET engineers in small scrum teams to deliver high quality test automation for products in a timely, cost-effective, and predictable manner.

Essential Duties and Responsibilities:
 **  Create, maintain, and execute comprehensive suites of test automation against our cloud service platform
** Enhance and extend the test automation framework, system, and libraries to test complex, large-scale, distributed systems
** Analyze and troubleshoot test failures, submit defects, and communicate test results.
** Participate in day-to-day SDET activities, including the continuous integration and Continuous Delivery build environment.
** Collaborate with developers to define and develop reusable test frameworks
** Build tools that make our Development and SDET teams more productive

Skills :
** Proficient one of the following: Java, Java Script, and Python
** Knowledge/Exposure to testing and automating integration around cloud service platforms REST APIs with Java as the development language.
** Exposure to agile development methodology
** Knowledge of SQL skills
** Experience with Version Control Systems - Git, Github, Bitbucket.
** Good verbal and written communication skills.
** Ability to work in and execute tests within a Windows, Unix/Linux environment
