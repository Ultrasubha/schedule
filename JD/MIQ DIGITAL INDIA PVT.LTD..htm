Job Profile

1.) Designation : Software Engineer Intern :- Role purpose
** As a Software Engineer Intern, you will collaborate with our senior engineers on assignments.
** You will have the opportunity to develop software components and have ownership of them.
** You will have the opportunity to utilize a variety of latest web application frameworks such as React, Big Data technologies, micro services development etc. You will be part of agile teams that utilize CI and CD pipelines to speed up the products’ time to market.
Key Responsibilities
** Developing software components.
** Debugging and testing code.
** Documentation.
** Researching, investigating and fixing a wide range of technical issues.
** Collaborating with senior engineers.
** Approaching problems and creating solutions.
** Proactively learning about new technologies.

Qualifications & Experience
** Knowledge in development with any Object-Oriented Language (Java, C++).
** Excellent grasp of at least one language.
** Excellent understanding of algorithm & data structures.
** Good knowledge of the Linux environment.
** Basic understanding of multi-tier web applications, hands-on across layers of the stack is a plus.
** Basic knowledge of SQL and NoSQL databases is a plus
** Basic knowledge of Big Data technologies and Distributed systems is a plus.
** Experience with cloud environments such as AWS, Google Cloud or Azure is a plus.

2.) Designation:  Security Engineer Intern Role purpose
** As a Security Engineer Intern you would learn about the security ecosystem across applications, infrastructure and cloud. 
** You will be knowledgeable around many different pieces of technology such as the Pen Test, Security tools across Cloud, Compliance & Certifications, Vulnerability identification & resolution techniques. 
** As a Security Engineer you will need to be a good communicator as you will interact with various stakeholders on a regular basis.

Key Responsibilities
** Work with fellow Security engineers.
** Assist senior resources in building and debugging security issues
** Manage existing automations and alerting for security vulnerabilities.** Write scripts which automate repetitive tasks.
** Monitor and Action on Cloud and Infra within the Org
** Identify and suggest resolutions which appear within applications for security related code.
** Security operations, network monitoring, threat analysis and investigation response solutions
** Active threat hunting and adversary tracking.

Qualifications & Experience
** Critical thinking and problem solving.
** Fast learner with different technologies.
** Good knowledge of the Linux environment.
** Basic understanding of Cloud and Infrastructure
** Basic knowledge of how web application work

3) Designation :  DevOps Intern :-Role purpose
** As a DevOps Intern you would learn about setting up infrastructure and pipelines to assist developers. 
** You will be knowledgeable around many different pieces of technology such as CI/CD, Terraform, Monitoring, Docker and Containerised Ecosystem.
** As a DevOps Engineer intern you will need to be a good communicator as you will interact with various stakeholders on a regular basis.

Key Responsibilities
** Work with fellow DevOps engineers.
** Assist senior resources in building CI/CD pipelines
** Existing Build pipeline management and optimization.
** Write scripts which automate repetitive tasks.
** Monitor and Action on Cloud and Infra within the Org
** Identify and fix bugs which appear within devops related code.
** Ensure pipelines are not creating bottlenecks for rapid deployment of code.
** Provide support to software developers who use the CI/CD pipelines.

Qualifications & Experience:
** Critical thinking and problem solving.
** Fast learner with different technologies.
** Good knowledge of the Linux environment.
** Basic understanding of Cloud and Infrastructure
** Basic knowledge of any one of scripting languages like Python/Shell etc
