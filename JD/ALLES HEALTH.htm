Job ProfileBackend Developer (React Native)
Responsibilities:
** Design, develop, and maintain backend services and APIs using Node.js and MySQL, ensuring high performance, responsiveness, and security.
** Collaborate with frontend developers, UI/UX designers, and other stakeholders to understand requirements, propose solutions, and implement backend features that  align with the overall product vision.
** Develop robust and maintainable database schemas, query optimization strategies, and data models using MySQL.
** Implement authentication, authorization, and data validation processes to ensure the security and integrity of user data and system resources.
** Integrate third-party APIs and services as needed to enhance the functionality of our platform.
** Write clean, maintainable, and well-documented code following established coding standards and best practices.
** Participate in code reviews, provide constructive feedback to peers, and continuously improve code quality.
** Troubleshoot and resolve backend issues, identify performance bottlenecks, and optimize application performance.
** Collaborate with the DevOps team to deploy and maintain backend services in cloud environments (e.g., AWS, Azure) and ensure seamless scaling and monitoring.
** Stay up-to-date with the latest trends and advancements in backend development, particularly in the Node.js and MySQL ecosystems.

Qualifications:
** Bachelor's degree in Computer Science, Engineering, or a related field (or equivalent experience).
** Proficiency in Node.js for building scalable and efficient server-side applications.
** Experience with designing and implementing RESTful APIs.
** Strong understanding of database systems (especially MySQL).
** Knowledge of AWS services for integration, deployment and management, such as EC2, Lambda, RDS, Aurora, API Gateway, Cognito and S3.
** Hands-on experience with version control using Git and GitHub repository management.
** Ability to write clean, maintainable, and well-documented code.
** Knowledge of testing frameworks like Mocha or Jest for unit and integration testing. (Optional)
** Proficiency in handling authentication and authorization mechanisms.
** Understanding of caching mechanisms and performance optimization strategies.
** Experience with continuous integration and deployment (CI/CD) pipelines.
** Ability to work with frontend developers to integrate APIs and implement new features.
** Familiarity with serverless architecture and microservices principles. (Nice to have)
** Strong problem-solving skills and the ability to analyze and debug complex backend issues.
** Excellent communication skills and the ability to work effectively in a collaborative, fast-paced environment.
** Previous experience in the healthcare or telemedicine industry is a plus.
